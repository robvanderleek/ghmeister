# Quickstart

## Installation

To install GH Meister in an isolated Python environment using
[pipx](https://pipx.pypa.io/stable/) run:

```shell
pipx install ghmeister
```

# Development

## Release to PyPi

1. Create a distribution build:
    ```shell
    poetry build
    ```

2. Publish to PyPi:
    ```shell
    poetry publish
    ```
